# Mini
 Tema para RetroPie inspirado en consolas "mini". Incluye nes, snes J, snes U, Psx
# Tema Mini para RetroPie

## Instalación rápida

1. **Descarga** el tema desde GitHub (`Code` → `Download ZIP`) y descomprímelo, o clónalo con Git.
2. **Copia** la carpeta descargada (por ejemplo, `mini-main` o `mini`) en:

   ```
   ~/.emulationstation/themes/
   ```

   Puedes usar este comando desde la terminal si descargaste el ZIP:
   ```bash
   mv mini-main ~/.emulationstation/themes/
   ```

3. **Copia el script `Customize mini.sh` a la carpeta del menú de RetroPie**:

   ```bash
   cp "Customize mini.sh" ~/RetroPie/retropiemenu/
   ```

   > Si usas otro usuario que no sea `pi`, reemplaza el nombre de usuario en la ruta, o simplemente usa `~/RetroPie/retropiemenu/`.

4. **Desde el menú de RetroPie**, entra a Retropie Menu y ejecuta el script de personalización.

   O, si prefieres terminal:
   ```bash
   bash ~/RetroPie/retropiemenu/Customize\ mini.sh
   ```
   *(Asegúrate de tener instalado `dialog`)*

5. **Elige la plantilla** que prefieras en el menú.

6. **Reinicia EmulationStation** para aplicar los cambios.

---

**¡Listo! El script detectará automáticamente la carpeta del tema, sin importar su nombre.**
